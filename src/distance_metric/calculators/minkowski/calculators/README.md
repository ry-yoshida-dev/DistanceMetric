# calculators

## Components

| Path | Description |
|------|-------------|
| `manhattan.py` | L1 (Manhattan) distance. |
| `euclidean.py` | L2 (Euclidean) distance. |
| `minkowski.py` | General Lp distance with a user-set exponent (`norm_order`) on the instance. |
| `chebyshev.py` | L∞ (Chebyshev) distance. |
| `squared_euclidean.py` | Squared Euclidean distance (no final square root). |
